Canvas Layering

 
 

Learning Intentions:

  1. Use multiple <canvas> elements for layered game graphics
  2. Organise game assets across different canvas layers

Success Criteria:

Students will be able to:

  1. Create a game interface with multiple canvas layers for different game elements
  2. Implement proper drawing order for game elements across different canvas contexts
  3. Manage separate canvas contexts for:
    1. Background elements (water, road)
    2. Moving obstacles (logs, cars)
    3. Player character (frog)
    4. Foreground elements (grass, score)
